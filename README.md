# Mouseion OS

The self-reading library. A personal research instrument that doesn't just store source material — it reads it.

Website: https://martijnaslander.github.io/mouseion-os/

## Structure

- `docs/` - GitHub Pages site
- `docs/alexandrie/` - Alexandria tradition subpages
- `docs/methode/` - Method subpages
- `docs/blog/` - Blog posts
- `docs/blog/feed.xml` - RSS feed

## Local development

Open `docs/index.html` in a browser.

## Deploy

Push to `main` branch. GitHub Pages serves from `docs/` folder.
